# Empty Workspace

A blank Worka workspace template used for intent-driven workspace creation.

This repository is a Worka workspace template. Worka indexes `worka-template.toml`, shows the
screenshots in the marketplace, and materializes the app, PersonalDB schema, requirements, and
specification into a new workspace when a user chooses the template.

The template is intentionally ordinary source code. There is no hidden demo bypass: prepared demo
workspaces are created by going through normal Worka template materialization.
