Workspace-local packs and generated app MCP servers live here. Each pack is a Cargo crate and rmcp server using rmcp::transport::worka::WorkaTransport and WORKA_BROKER_SOCKET. Marketplace or built-in packs may also be materialized here when the controller provides them. When docs/specification.json declares generated_app_mcp_server.required, copy the templates under .worka/templates/generated-app-mcp/ into the declared crates/packs/<app-domain-pack> path and replace the placeholders with the declared tool names, request models, response models, PersonalDB groups, and audit events.
