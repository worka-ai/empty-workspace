# Requirements - Empty Workspace

## Outcome

The workspace must show useful operational state immediately after materialization and must make
agent work visible as a first-class part of the app.

## Required capabilities

- Persist workspace-owned records through brokered PersonalDB access.
- Route external integration events into the AI team through connected packs.
- Let agents perform domain actions through generated app MCP tools.
- Show inline elicitations when a human decision or file input is required.
- Keep a central audit trail linking event, agent, tool call, and resulting app state.
